## 4 · How a call actually flows

1. Pick a lead from the Lead Picker → tick the checkbox → click
   **Start calling →**
2. The Call Workspace opens. You'll see the prospect's info (business
   name, phone, city, what we know about them), the **call script** in
   the middle, **objections + responses** on the right, and **outcome
   buttons** at the bottom.
3. Dial the number. Use the script as a guide — don't read it
   word-for-word.
4. When the call ends (or if voicemail), **click the matching outcome
   button**:
   - 🤝 **Booked meeting** (`answered_call_scheduled`) — the win we want
   - 📤 **Sent preview** / 📤 **Sent audit** / 📤 **Sent callback** —
     they engaged but didn't book on-the-call
   - 🙅 **Not interested** — they passed
   - 📞 **Voicemail** — left one
   - 🤷 **No answer** / **Wrong number**
   - 🛑 **Take me off the list** — TCPA compliance; we never call again
5. Add notes if useful (optional — short context helps Ben later).
6. Click **Next prospect →** when ready.

**Every outcome you log shows up in the productivity strip within 60s.**
If you don't see it move, refresh the page once. If still nothing,
text Ben.
