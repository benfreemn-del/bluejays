# Bloodlines — Character Portraits Brief

Hand-off document for image generation. 13 character portraits for the
Bloodlines fantasy saga client showcase at `/clients/bloodlines`.

## Delivery spec

- **Format:** PNG, square 1024×1024 (preferred). 3:4 portrait acceptable.
- **Composition:** Head-and-shoulders. Head positioned in upper third of
  frame so a circular face crop centered at ~22% from top reads cleanly.
- **Filenames:** Match exactly — `<id>.png`, lowercase, hyphens preserved.
- **Drop location on return:** `bluejays/public/images/clients/bloodlines/characters/`
- **Component is already wired** — drop the files into the directory
  above and refresh the page. The 13 monogram fallbacks auto-replace
  with the portraits. No code change needed on return.

## Saga context (for the agent)

Bloodlines is a fantasy saga by Preston James Hunsaker (books: *Lineage
of Fire*, *House of the Rose*). Setting: kingdom of Annarose. Core
conflict: two friends from the Uplands — Sopher and Proph — discover
ancient bloodlines and a tech-magic system called the elletas. Five
named elletas: pyrelle (fire), stonewake (earth), tideborn (water),
skyveil (air), lumengarde (light). Five factions: Royal Guard / Wilted
Rose / Archivists / Tags Merchants / Elleta-Bonded. Wilted-rose tattoo
is the bloodline sigil.

Recommended for readers 11+ but the visual register is mature
sword-and-sorcery, not YA fluff. Think Brom paintings, Donato Giancola
covers, weathered leather, golden firelight on dark surfaces.
